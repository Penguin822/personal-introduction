🤖 實作 AI Agent：專屬 LINE 機器人「小龍蝦」

Node.js OpenClaw OpenAI GPT API LINE Official Account ngrok Git

這是我自己動手做的一個 Side Project，利用 OpenClaw 框架和 GPT 模型，開發了一個可以理解自然語言並執行任務的 LINE 機器人（我叫它小龍蝦）。為了讓本機端的 Node.js 服務能跟 LINE API 溝通，我使用了 ngrok 來建立網路隧道，並全程使用 Git 進行版本控管。

🔒 關鍵亮點：VM 沙盒資安隔離

因為 AI Agent 有能力自己生成並執行程式碼，如果直接架在實體主機上，會有很高的被駭風險（例如：惡意 Prompt Injection 導致主機資料被刪除）。
所以我特別**在虛擬機 (VM) 中建置它的執行環境**。這樣做形同建立了一個沙盒 (Sandbox)，就算 Agent 執行了危險指令，也只會被限制在 VM 裡面，完全不會影響到我的外部主機，確保了系統的安全性。

👩‍💻 Microsoft Coding Angels 計畫

參加微軟舉辦的 Coding Angels 計畫，與許多熱愛科技的女同學一起交流學習。在這裡我最大的收穫是接觸到了 GitHub Copilot。

我學會了如何把 AI 當作結對編程 (Pair Programming) 的夥伴，透過良好的註解習慣引導 Copilot 生成程式碼，這大幅改變了我寫 Code 的節奏，讓我能把更多時間花在思考架構，而不是糾結於語法細節。

✨ 日常開發的 AI 協作 (Claude)

除了實作 AI 應用，我平常也是重度的 AI 工具使用者。在日常開發中，我會將 Claude 作為我的技術顧問：

  • 架構發想：在開新專案前，我會跟 Claude 討論資料庫 Schema 怎麼設計比較合理。
  • 程式碼重構：遇到寫得太肥大的 Function，我會請它提供 Clean Code 的修改建議。
  • 快速除錯：把難解的報錯 Log 丟給它，可以幫我省下大量爬 StackOverflow 的時間。

這些經驗讓我培養了很好的 Prompt Engineering (提示詞工程) 能力，知道怎麼精準描述問題，讓 AI 成為我開發上最強的輔助。
